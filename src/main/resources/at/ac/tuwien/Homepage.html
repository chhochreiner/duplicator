How to get started

Import sample data

Go to Batch import and import the file testdata.csv file.

Import sample templates

Go to Import template and import the sample templates file.

Execute the data

Then you can generate Selenium tests, that can be executed via the specific runner for this app.

Import external data

You can also import profile via this API - Call:
There are some required parameters: birthday, prename, surname, email and password If you want to import a image, add an additional key value pair with avatar and the absolute URL/Filepath of the image
e.g:
http://localhost:8080/wicket/bookmarkable/at.ac.tuwien.service.impl.ExternalProfile?birthday=01.01.2011&prename=Max&surname=Musermann&email=mm@gmail.com&password=asdf
